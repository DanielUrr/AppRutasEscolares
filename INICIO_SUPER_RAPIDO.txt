┌────────────────────────────────────────────────────────────────────┐
│  🚀 INSTALACIÓN EXPRESS                                            │
└────────────────────────────────────────────────────────────────────┘

ANDROID (15 minutos):
─────────────────────────────────────────────────────────────────
1. Descargar: archivos_faltantes.zip
2. Extraer al mismo nivel que transporte_escolar
3. Ejecutar:
   cd transporte_escolar
   bash ../archivos_faltantes/copiar_archivos.sh
4. Descargar google-services.json de Firebase Console
5. Colocar en: android/app/google-services.json
6. Ejecutar:
   flutter clean
   flutter pub get
   flutter run


iOS (20 minutos - Solo Mac):
─────────────────────────────────────────────────────────────────
1. Descargar: archivos_ios.zip
2. Extraer al mismo nivel que transporte_escolar
3. Ejecutar:
   cd transporte_escolar
   bash ../archivos_ios/setup_ios.sh
4. Descargar GoogleService-Info.plist de Firebase Console
5. Colocar en: ios/Runner/GoogleService-Info.plist
6. Ejecutar:
   flutter run


┌────────────────────────────────────────────────────────────────────┐
│  🔥 CONFIGURAR FIREBASE (Ambas plataformas)                       │
└────────────────────────────────────────────────────────────────────┘

1. Ir a: https://console.firebase.google.com/
2. Crear o seleccionar proyecto
3. Agregar app Android:
   - Package: com.ejemplo.transporte_escolar
   - Descargar: google-services.json
4. Agregar app iOS:
   - Bundle ID: com.ejemplo.transporteEscolar
   - Descargar: GoogleService-Info.plist
5. Editar lib/main.dart con tus credenciales
