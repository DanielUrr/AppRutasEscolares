PASO 1: Instalar Dependencias
──────────────────────────────────────────────────────────────
cd transporte_escolar_completo
flutter pub get

# Solo para iOS (Mac):
cd ios
pod install
cd ..


PASO 2: Configurar Firebase
──────────────────────────────────────────────────────────────
1. Ve a: https://console.firebase.google.com/
2. Crea un proyecto nuevo
3. Agrega app Android:
   - Package: com.ejemplo.transporte_escolar
   - Descarga: google-services.json
   - Coloca en: android/app/google-services.json
4. Agrega app iOS:
   - Bundle ID: com.ejemplo.transporteEscolar
   - Descarga: GoogleService-Info.plist
   - Coloca en: ios/Runner/GoogleService-Info.plist
5. Edita lib/main.dart (línea 26) con tus credenciales


PASO 3: Habilitar Servicios en Firebase
──────────────────────────────────────────────────────────────
1. Authentication → Enable Email/Password
2. Firestore Database → Create database
3. Cloud Messaging → Configure


PASO 4: Ejecutar
──────────────────────────────────────────────────────────────
flutter run

Error: "google-services.json not found"
→ Descárgalo de Firebase Console
→ Colócalo en: android/app/google-services.json

Error: "GoogleService-Info.plist not found"
→ Descárgalo de Firebase Console
→ Colócalo en: ios/Runner/GoogleService-Info.plist

Error: "Firebase not initialized"
→ Verifica credenciales en lib/main.dart

Error: "CocoaPods not installed" (iOS)
→ sudo gem install cocoapods
→ pod setup
